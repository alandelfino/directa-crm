# Skill — Composição com shadcn/ui

Use quando criar ou alterar componentes visuais.

## Fluxo

1. Pesquise no MCP Context7 se existe componente equivalente ou semelhante no `shadcn/ui`.
2. Se existir, instale com o comando oficial do `shadcn/ui`.
3. Use o componente como base.
4. Adapte apenas o necessário.
5. Se não existir, crie componente personalizado.
6. Sempre que possível, componha usando componentes existentes do `shadcn/ui`.

## Componentes Base

Considere usar:

- `Button`
- `Card`
- `Dialog`
- `Input`
- `Select`
- `Badge`
- `Tabs`
- `Table`

## Regras

- Evite componentes visuais totalmente manuais.
- Preserve padrões visuais existentes.
- Preserve acessibilidade.
- Preserve responsividade.
- Preserve consistência com o design system.
